# Learning Circles Guide – TechAscend Cohort 2

Learning Circles are a core part of the TechAscend structure.

They are mandatory.

---

## Purpose

Learning Circles exist to:

- Create accountability
- Encourage peer support
- Strengthen understanding
- Prevent silent struggles
- Build professional collaboration habits

---

## Structure

Each circle contains 3–5 participants.

Sessions occur every Wednesday.

Duration: 60–90 minutes.

---

## Recommended Session Flow

1. Progress Updates (10–15 minutes)
   Each member shares:
   - What they completed
   - What is still pending
   - Where they are stuck

2. Collaborative Debugging (30–40 minutes)
   - Review code together
   - Solve problems as a group
   - Clarify misunderstandings

3. Concept Clarification (10–20 minutes)
   - Discuss confusing topics
   - Share resources

4. Commitment Setting (5–10 minutes)
   Each member states:
   - What they will complete before Sunday

---

## Responsibilities

Each circle must:

- Appoint a rotating facilitator
- Ensure everyone speaks
- Post a short summary in Slack

Summary format:

Learning Circle Summary – Week X

Members Present:
Topics Discussed:
Key Blockers:
Commitments for Next Week:

---

## Expectations

- Attendance is required.
- Preparation is required.
- Respectful communication is required.

If you cannot attend:
- Inform your circle in advance.

Discipline builds momentum.
