# Week 1 – Grading Criteria (AI Track)

Total: 100 Points

---

## 1. Environment & Setup (20 points)

- Repository correctly named (5)
- Python file or notebook present (5)
- Code runs without major errors (10)

---

## 2. Python Fundamentals (30 points)

- Variables implemented correctly (5)
- Loop implemented correctly (5)
- Function implemented correctly (5)
- List manipulation correct (5)
- Dictionary usage correct (5)
- Mini problem completed logically (5)

---

## 3. Code Quality (15 points)

- Clean formatting (5)
- Readable structure (5)
- Logical flow (5)

---

## 4. Git Usage (15 points)

- Minimum 3 commits (5)
- Clear commit messages (5)
- Progressive improvement shown (5)

---

## 5. README Reflection (10 points)

- All required sections completed (5)
- Thoughtful reflection (5)

---

## 6. Professionalism (10 points)

- Organized repository
- Proper naming
- Submission format followed

---

## Automatic Deductions

- Wrong repo name
- Missing README
- Single commit only
- Code does not run
- Incorrect Slack format
- Late submission

Consistency and structure are key in AI development.
