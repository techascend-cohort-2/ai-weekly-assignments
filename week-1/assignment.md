# Week 1 – AI Track
## Python Foundations & Git Setup

### Objective
Set up your AI development environment and demonstrate basic Python competency.

This week is about:
- Python setup
- Git discipline
- Core programming fundamentals

---

## Step 1 – Environment Setup (Mandatory)

Install:

- Python (latest stable version)
- VS Code OR Jupyter Notebook
- Git

Verify:

- You can run a Python file
- You can install packages
- You can push to GitHub

---

## Step 2 – Python Fundamentals

Create a Python file OR Jupyter Notebook that includes the following exercises:

### Required Exercises

1. Variables:
   - Declare variables of different types (int, float, string)

2. Loops:
   - Write a loop that prints numbers 1 to 10

3. Functions:
   - Create a function that takes two numbers and returns their sum

4. Lists:
   - Create a list of 5 items
   - Loop through the list and print each item

5. Dictionary:
   - Create a dictionary representing a person (name, age, city)
   - Print values properly

6. Mini Problem:
   - Write a short program that:
     - Asks for user input
     - Processes it
     - Prints a result
